# Annalink Blockchain Documentation

## Introduction

Annalink is a full-featured blockchain implementation built from scratch in Python. It includes all essential components of a production blockchain: persistent storage, P2P networking, wallet management, consensus mechanism, and APIs.

## Features

- ✅ Persistent storage (SQLite)
- ✅ P2P networking (TCP)
- ✅ Wallet management (ECDSA + Base58)
- ✅ Proof-of-Work consensus
- ✅ Transaction validation
- ✅ Block mining
- ✅ REST API
- ✅ CLI interface
- ✅ Configuration management
- ✅ Comprehensive testing
